# Currency Converter App

A web application to convert currency values from one currency to another in real-time using exchange rates.

### Live Demo : https://currency-converter-pi-ashy.vercel.app/
---

## Features

1. **Live Currency Conversion**:
   - Fetches real-time exchange rates.
   - Converts amount from one currency to another.

2. **User-Friendly Interface**:
   - Simple and intuitive design for easy usage.

3. **Responsive Design**:
   - Works seamlessly across various screen sizes and devices.

---

## Project Structure

### Files:
- `App.js`: Main React component handling UI and logic.
- `App.css`: Manages the layout and styles.
- `index.js`: Entry point for the React application.

---

## Installation and Usage

1. Clone this repository or download the source files.
2. Install dependencies using:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm start
   ```
4. Open `http://localhost:3000` in a browser.
5. Enter an amount, select currencies, and get the converted value.
